📚 Referencia rapida ZPL
Estrutura basica:
^XA             → inicio do label
^CI28          → encoding UTF-8
^PWn          → largura em dots (mm × 8)
^LLn          → altura em dots (mm × 8)
^FOx,y       → posicao do campo
^A0N,h,w    → fonte (h=altura, w=largura)
^FBw,1,0,C   → texto centralizado (largura w)
^FDtexto^FS  → conteudo do campo
^BQN,2,n     → QR Code (n=tamanho 1-10)
^FDQA,dado^FS → dado do QR Code
^XZ             → fim do label
